# Python-Project-Data-Processing-and-Analysis
Python project covering problem solving, NumPy analysis, pandas data processing, and data visualization using real datasets.

## Overview
This project covers a wide range of Python concepts including problem-solving, data structures, functional programming, NumPy-based numerical analysis, and data analysis using pandas and Matplotlib. It demonstrates the ability to write clean logic, process data, and generate insights from datasets.

---

## Project Scope

The project is divided into three main sections:

1. Core Python Problem Solving  
2. Numerical Computing using NumPy  
3. Data Analysis and Visualization using pandas and Matplotlib  

---

## Core Python Concepts

### Functional Logic
- Discount calculation engine with conditional rules
- Parking fee calculator with tiered pricing logic
- Password strength evaluation

### String and Data Processing
- Extracting structured data from strings
- URL parameter parsing into dictionaries
- Anagram checking
- Character frequency analysis

### Data Structures
- Grouping data using dictionaries
- Finding top performers based on computed averages
- Removing duplicates while preserving order

### Advanced Logic
- Recursive flattening of nested lists
- Data cleaning and transformation tasks

## NumPy – Numerical Analysis

Performed operations on transaction data:
- Calculated mean and standard deviation

## Data Analysis and Visualization

### Revenue Analysis (sales.csv)
- Converted date columns into datetime format
- Aggregated revenue by month and region
- Created:
  - Line chart for monthly revenue trends
  - Bar chart for regional revenue comparison
- Derived insights from visual trends

### Student Performance Analysis (students.csv)
- Loaded dataset using pandas
- Calculated average marks per subject
- Filtered students based on marks and attendance criteria
- Created a Result column using conditional logic
